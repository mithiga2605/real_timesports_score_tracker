Project Title: Real-Time Sports Score Tracker

Description:
The Real-Time Sports Score Tracker is a simple C++ console-based application designed to display and update scores of two teams during a match. Users can input team names and continuously update scores through an interactive menu. The program shows the current score in real time and declares the winner at the end. This project demonstrates basic concepts of data structures, loops, conditional statements, and user interaction in C++.
